# Customer Journey Analytics MCP server

The Customer Journey Analytics MCP server lets LLM clients interact with your components and data in Customer Journey Analytics. Once connected, you can retrieve data views, dimensions, metrics, segments, and run reports through natural language prompts.

## Endpoint

```
https://cja-mcp.adobe.io/mcp
```

## Capabilities

* Discover data views and their configurations
* Find and describe dimensions, metrics, and calculated metrics
* Create and run ranked and trended reports
* Manage segments, audiences, and date ranges
* Explore workspace projects

See the [tool reference](reference.md) for a complete list of available tools and their parameters.
